# Webscape

Webscape is a web-based platform developed as the final project for our capstone, designed to address the challenge of user engagement in online learning environments. Built with a focus on interactivity and community, Webscape leverages a MySQL database to deliver a dynamic and engaging experience for users.

## Project Background

This application was created as part of our capstone project, where we identified low engagement as a key issue in digital education. Our goal was to design and implement a solution that not only encourages participation but also fosters a sense of achievement and competition among users.

## Key Features

- **Interactive Code Battles:** Users can participate in real-time code challenges against peers, promoting friendly competition and skill development.
- **Quiz Battles:** Engage in quiz-based competitions to test and expand knowledge in a fun, gamified format.
- **Badges & Rankings:** Earn badges for achievements and climb the rankings leaderboard, providing motivation and recognition for active participation.
- **Lesson Comments:** Users can comment on lessons, ask questions, and interact with the community, making learning more collaborative.
- **Admin Tools:** Includes features for managing users, content, and monitoring platform activity.

## Technology Stack

- **Backend:** Python (Flask)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Other:** Google Analytics integration, custom static assets, and a modular template system.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Luxanna22/webscape.git
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure your database:**
   - Set up a MySQL database and update your connection settings in the app configuration.
4. **Run the application:**
   ```bash
   python app.py
   ```
